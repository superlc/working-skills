# 响应式测试检查清单

## 通用检查项（所有端适用）

### 布局完整性
- [ ] 无水平滚动条溢出（除非设计意图如此）
- [ ] 所有内容可见，无被裁剪的文字或元素
- [ ] 页面无空白间隙或重叠区域
- [ ] 背景图/装饰元素正确适配
- [ ] 页面最小高度铺满视口

### 文字可读性
- [ ] 文字未溢出容器
- [ ] 文字大小在当前视口下可读（mobile ≥ 12px）
- [ ] 长文本正确换行或省略（text-overflow: ellipsis）
- [ ] 行宽合理（每行不超过 80 字符为佳）

### 交互可用性
- [ ] 可点击元素尺寸合理（mobile 点击目标 ≥ 44×44px）
- [ ] 导航菜单在当前视口下可用
- [ ] 表单输入框宽度合理
- [ ] 弹窗/模态框在当前视口下可正常显示和关闭

### 图片与媒体
- [ ] 图片未变形（保持宽高比）
- [ ] 图片清晰度与视口 DPR 匹配
- [ ] 图片未溢出容器

---

## Desktop 专项检查（≥ 1440px）

- [ ] 内容区域宽度有最大限制（避免超宽拉伸）
- [ ] 多列布局正确呈现
- [ ] 侧边栏（如有）正常显示
- [ ] Hover 效果正常工作
- [ ] 表格数据完整展示（无不必要的横向滚动）

## Tablet 专项检查（768–1024px）

- [ ] 导航从桌面模式切换为合适的平板布局（如折叠菜单）
- [ ] 多列布局合理调整（如 3→2 列或 4→2 列）
- [ ] 侧边栏折叠或转为抽屉模式
- [ ] 卡片/网格间距合理
- [ ] 表格在该宽度下处理合理（横向滚动 / 列折叠 / 卡片化）

## Mobile 专项检查（≤ 414px）

- [ ] 导航转为汉堡菜单或底部标签栏
- [ ] 布局变为单列
- [ ] 按钮和输入框宽度 ≥ 容器宽度或合理定位
- [ ] 底部固定操作栏不遮挡内容
- [ ] 触控目标间距足够（≥ 8px）
- [ ] 表格转为卡片式或可横向滚动
- [ ] 弹窗宽度适配屏幕（全屏或近全屏）
- [ ] 长列表有虚拟滚动或分页

---

## 跨端一致性检查

- [ ] 核心功能在三端均可操作完成
- [ ] 数据展示完整性一致（不丢失关键信息）
- [ ] 品牌一致性（颜色、字体、图标风格统一）
- [ ] 加载态和错误态在三端均有处理

---

## 常见响应式问题模式

| 问题 | 症状 | 常见原因 |
|------|------|---------|
| 水平溢出 | 出现横向滚动条 | 固定宽度元素未设 `max-width: 100%` |
| 文字溢出 | 文字超出容器 | 缺少 `overflow-wrap: break-word` |
| 触控困难 | 按钮/链接难以点击 | 点击区域 < 44px |
| 内容裁剪 | 文字/图片被截断 | `overflow: hidden` + 固定高度 |
| 空白断层 | 页面中间出现大块空白 | 绝对定位或固定高度在小视口下失效 |
| 元素重叠 | 两个元素叠在一起 | `position: absolute` 未响应式适配 |
| 图片变形 | 图片拉伸或压缩 | 缺少 `object-fit: cover/contain` |
| 列布局挤压 | 多列内容挤在一起 | 未设断点让多列变单列 |
